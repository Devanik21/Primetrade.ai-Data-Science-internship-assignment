# Cryptocurrency Price Prediction

## Project Overview
This project focuses on analyzing cryptocurrency market data and predicting future price movements using historical price data and the Crypto Fear & Greed Index.

## Project Structure
```
Debanik_Debnath/
├── data/                    # Data files
│   ├── raw/                 # Raw data (never modify)
│   └── processed/           # Processed data
├── notebooks/               # Jupyter notebooks for analysis
├── src/                     # Source code
│   ├── data/                # Data processing scripts
│   ├── models/              # Model training and evaluation
│   └── visualization/       # Visualization scripts
├── requirements.txt         # Project dependencies
└── README.md                # Project documentation
```

## Setup
1. Clone the repository
2. Create a virtual environment: `python -m venv venv`
3. Activate the environment:
   - Windows: `venv\Scripts\activate`
   - Unix/MacOS: `source venv/bin/activate`
4. Install dependencies: `pip install -r requirements.txt`

## Usage
1. Place your data files in the `data/raw/` directory
2. Run the Jupyter notebooks in the `notebooks/` directory

## Data Sources
- Historical cryptocurrency price data
- Crypto Fear & Greed Index data
